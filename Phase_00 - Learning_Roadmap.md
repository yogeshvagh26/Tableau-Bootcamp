## Learning Roadmap

### Phase 1: Tableau Fundamentals

* Introduction to Tableau
* Business Intelligence Concepts
* Installing and Setting Up Tableau Desktop
* Tableau Interface Overview
* Understanding Workbooks, Worksheets, Dashboards, and Stories
* Connecting to Data Sources
* Live Connections vs Extracts
* Tableau Workflow

### Phase 2: Data Preparation

* Connecting to Excel, CSV, SQL Databases, and Cloud Sources
* Understanding Data Types
* Data Interpreter
* Cleaning Data
* Data Joins
* Data Unions
* Data Blending
* Pivoting Data
* Managing Metadata

### Phase 3: Data Modeling and Calculations

* Dimensions and Measures
* Discrete vs Continuous Fields
* Calculated Fields
* Basic Calculations
* String Functions
* Number Functions
* Date Functions
* Logical Functions
* Aggregate Functions
* Table Calculations
* Level of Detail (LOD) Expressions

### Phase 4: Data Visualization

* Tables and Crosstabs
* Bar Charts
* Line Charts
* Area Charts
* Pie Charts
* Scatter Plots
* Maps
* Heat Maps
* Tree Maps
* Highlight Tables
* Bubble Charts
* Dual Axis Charts
* Combined Charts

### Phase 5: Interactive Dashboards

* Building Dashboards
* Dashboard Layouts
* Containers
* Filters
* Actions
* Parameters
* Navigation Buttons
* Tooltips
* Dynamic Titles
* Device Layouts

### Phase 6: Advanced Tableau

* Advanced Calculated Fields
* Advanced LOD Expressions
* Advanced Table Calculations
* Set Actions
* Parameter Actions
* Dashboard Performance Optimization
* Data Source Optimization
* Tableau Prep Basics

### Phase 7: Tableau Server and Cloud

* Introduction to Tableau Server
* Tableau Cloud
* Publishing Workbooks
* Sharing Dashboards
* User Roles and Permissions
* Scheduling Extract Refreshes
* Collaboration Features
* Governance Best Practices

### Phase 8: Dashboard Design Best Practices

* Choosing the Right Chart
* Dashboard Design Principles
* Color Theory
* Typography
* User Experience (UX)
* Storytelling with Data
* Accessibility
* Mobile Dashboard Design

### Phase 9: Performance Optimization and Security

* Performance Recording
* Optimizing Calculations
* Optimizing Data Sources
* Row-Level Security
* Data Security Best Practices
* Tableau Performance Tuning

### Phase 10: Real-World Projects

Build practical projects to apply your knowledge, such as:

* Sales Performance Dashboard
* Executive KPI Dashboard
* HR Analytics Dashboard
* Financial Dashboard
* Customer Segmentation Dashboard
* Marketing Campaign Dashboard
* Supply Chain Dashboard
* Healthcare Analytics Dashboard

## Learning Format

For every lesson, followed below structure:

1. Concept Explanation
2. Importance and Real-World Use Cases
3. Step-by-Step Demonstration
4. Practical Examples
5. Hands-on Exercises
6. Mini Quiz
7. Common Mistakes and Best Practices
8. Interview Questions
9. Assignment and Dashboard Projects
10. Summary and Revision Notes
